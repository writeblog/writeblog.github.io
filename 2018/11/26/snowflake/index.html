PHP 版本的 snowflake

由于 PHP 语言的特点，PHP中的数据用完就会销毁，下次再执行时又是全新的数据，这就导致在 PHP 中无效保存 上一次的数据 以及 持久存在的数字序列，为了解决这两个问题，我们可以使用 redis 来保存需要持久存在的数据。Snowflake 中有两个数据需要保存到 Redis ：上次生成的时间戳 和 序列ID。
